```powershell
xiuluohost.exe --cookie "xxx" --groupUrl "https://www.xiuluohost.com/product/zj-nat" --productName "KVM-ZS-NAT-A1" --buyWay "alipay"
```
参数说明
cookie:登录成功之后的token
groupUrl:产品组url
productName:产品名称
buyWay:支付方式 默认 balance 余额支付，此选项保证有充足的余额，alipay,wxpay
-h:获取详细说明
